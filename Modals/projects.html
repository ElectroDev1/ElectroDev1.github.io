Holemap

2022

✖

Holemap is a small library made for Gamemaker v2.3+, this library gives you functions to easily take tile layers present in a room and pack them in a buffer that can then be turned into a surface.

This allows for easy manipulation of tiles for various effects, the project in the repository shows various examples of how Holemap can be used.

Github

Surfs

2022

✖

Surfs is a library made for Gamemaker v2022.5+ that handles all the boring and monotonous parts of using surfaces in Gamemaker, with only a few simple functions and no need to use multiple events or extra variables.

Github

CCM

2022

✖

CircleColorMulti, or CCM in short, is a very small library that allows you to draw colored circles with more colors and more complex patterns than the linear interpolation between two colors that Gamemaker's default draw_circle_color provides.
CCM even comes with a set of functions that allow you to treat circles as objects
