SPRING ESCAPE ANNOUNCEMENT + FUTURE PROJECTS

17/07/2022

Earlier today, I finally announced the latest game I've been working on, Spring Escape, a little platformer about a piece of dirt that can dig into other pieces of dirt and move around. I've been working on this little game for a couple of months, and so far I've had a lot of fun working on it with the help of my friends, people really liked the trailer and I cannot wait to finally release the demo for the game.

But what's the deal with this game? Wasn't I already working on another one?
Yes, I was working on another one, Rocket Hat, and the answer is simple, I decided to hault work on that project for a while as I focus on this smaller and simpler project. I will restart work on Rocket Hat once I feel like I've improved my skills to deal with a project as ambitious as that and I have a clearer vision of what the project will be like. But for now, my only focus is Spring Escape.

I plan to work and expand on Spring Escape for around year, and then release it on Steam. I want to keep the scope small and focused on this one, so no, there won't be any convoluted stories or weird deep characters, just a simple game with fun gameplay. I already have plans for many future projects, and they will keep the same smaller scope to varying degrees as I eventually come back to working on Rocket Hat.
